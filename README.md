# yearbook-grabber
A quick script that will roll through a sequential number of images, download them, and bundle them into a PDF.

Each component is quick to use and easy to tie things together.

Usage of yearbook-grabber.sh

`$ yearbook-grabber.sh <number_of_images> <minimum number of digits for padding> <output_directory> <base_url>`

Usage of yearbook-binder.py

`$ python3 yearbook-binder.py <input directory> <Output Filename>.pdf "<PDF Title>" "<Name listed as Author>"`
